# System Design Prep

[![Live Site](https://img.shields.io/badge/live%20site-anuvik2401.github.io%2Fsystem--design--prep-6ee7b7?style=for-the-badge)](https://anuvik2401.github.io/system-design-prep/)

**➡️ [Open the live site](https://anuvik2401.github.io/system-design-prep/)**

Personal system design interview prep, built as a static site. Covers the delivery framework, core concepts, key technologies, common patterns, and 30+ full question breakdowns (Uber, WhatsApp, Google Docs, rate limiter, and more) — each with requirements, entities, API design, data flow, high-level design, and deep dives.

## Stack

Vanilla HTML + CSS + JS. No framework, no build step. Deployed via GitHub Pages from `main`.

## Structure

```
index.html              — landing page / topic index
topics/                 — foundations, patterns, tech, advanced, question breakdowns
assets/style.css         — shared styles
assets/nav.js            — sidebar, progress tracking, diagrams, interactions
```
